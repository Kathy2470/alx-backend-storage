# Redis Cache Project

## Overview

This project demonstrates the usage of Redis for caching data using a Python class. The `Cache` class interacts with Redis to store and retrieve data efficiently.

## Requirements

- Ubuntu 18.04 LTS
- Python 3.7
- Redis server
